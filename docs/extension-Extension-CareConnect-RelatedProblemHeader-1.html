Extension: Extension-CareConnect-RelatedProblemHeader-1

The official URL for this extension is: 

https://hl7.org.uk/fhir/StructureDefinition/Extension-CareConnect-RelatedProblemHeader-1

Status: draft
Extension maintained by: INTEROPen

A reference to another related problem header condition (target) whose relationship is defined by the relationship type code.

Context of Use

This extension may be used on the following element(s):

  • {"type"=>"Condition"}

Usage info

Usage:

  • Use this Extension: CareConnect-ProblemHeader-Condition-1
