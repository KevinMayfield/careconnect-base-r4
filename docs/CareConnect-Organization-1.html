StructureDefinition: CareConnect-Organization-1

The official URL for this profile is: 

https://hl7.org.uk/fhir/StructureDefinition/CareConnect-Organization-1

This profile defines an organisation administration details structure that includes core localisation concepts for use in an UK context.
